Документация к элементарному графическому редактору
===============================================

Введение
--------
Данное приложение представляет собой элементарный графический редактор, предназначенный для демонстрации построения отрезков с
использованием трёх алгоритмов: алгоритма ЦДА (Digital Differential Analyzer), целочисленного алгоритма Брезенхэма и алгоритма Ву
(антиалиасинг). Приложение позволяет выбирать требуемый алгоритм через выпадающее меню «Отрезок» на панели инструментов, а также
переключаться в режим отладки для пошагового отображения вычислительных таблиц, демонстрирующих процесс построения отрезка.

Алгоритмы построения линий
--------------------------
1. Алгоритм ЦДА (Digital Differential Analyzer):
   - Работает на основе вычисления приращений по осям X и Y.
   - Делит отрезок на равные отрезки по времени (шагам) и вычисляет координаты точки, приближающейся к идеальной линии.
   - Использует операции с плавающей точкой.
   - Отладочная таблица (режим отладки) содержит поля: Итерация, x (вещественное значение), y (вещественное значение), Отображение координаты
     (округлённые).

2. Алгоритм Брезенхэма:
   - Основан на использовании целочисленной арифметики для построения отрезка.
   - Использует понятие ошибки (difference) для определения направления движения по осям.
   - Позволяет быстро вычислять координаты пикселей, подбирающихся к идеальной линии.
   - Отладочная таблица демонстрирует шаги вычислений и содержит поля: Итерация, x, y, e (значение ошибки до корректировки),
     e′ (ошибка после корректировки), Отображение координаты.

3. Алгоритм Ву:
   - Применяет принцип антиалиасинга для сглаживания отрезков.
   - Вычисляет интенсивность (яркость) пикселей в зависимости от того, как близко к идеальной линии проходит отрезок.
   - Результатом является более гладкое изображение линии за счёт градации интенсивности.
   - Отладочная таблица для алгоритма Ву также состоит из полей: Итерация, x, y, e (значение, определяющее интенсивность для верхнего
     пикселя), e′ (интенсивность для нижнего пикселя), Отображение координаты.

Как использовать приложение
----------------------------
1. Запуск приложения:
   Запустите приложение командой «python main.py». Главное окно откроется с канвой, меню, панелью инструментов и строкой состояния.

2. Меню «Файл»:
   - Очистить экран: Удаляет все нарисованные объекты с канвы.
   - Выход: Завершает работу приложения.

3. Построение отрезков:
   - В режиме построения отрезков вам необходимо выполнить два клика:
     1) Первый клик задаёт начальную точку отрезка.
     2) Второй клик задаёт конечную точку.
   - После второго клика выбранный алгоритм построит линию между этими точками.

4. Режим отладки:
   - Нажмите кнопку «Отладка», чтобы включить или отключить режим отладки.
   - При включённом режиме отладки, после построения отрезка откроется отдельное окно с таблицей, в которой по шагам
     отображаются значения координат и ошибки, используемые алгоритмами для вычислений.
   - Для алгоритма ЦДА таблица выведет: Итерация, x, y, Отобр. координаты.
   - Для алгоритмов Брезенхэма и Ву таблица выведет: Итерация, x, y, e, e′, Отображение координаты.

5. Кнопка «Курсор»:
   - При нажатии на кнопку «Курсор» приложение переходит в режим перемещения, и построение отрезков временно отключается.

6. Меню «Помощь»:
   - Выберите пункт «Документация», чтобы открыть данное руководство пользователя в отдельном окне.

Требования к системе
---------------------
- Python 3.x
- Библиотека Tkinter (обычно входит в стандартную поставку Python)
- Файлы с исходным кодом: main.py, gui.py, intervals.py и данный файл документации (documentation.txt).

Заключение
-----------
Данное приложение служит учебным пособием для изучения алгоритмов построения отрезков в компьютерной графике.
Оно позволяет на практике увидеть различия между алгоритмами, а режим отладки помогает проследить внутреннюю логику каждого
из них. Надеемся, что приложение будет полезно для более глубокого понимания принципов работы с графическими примитивами.
